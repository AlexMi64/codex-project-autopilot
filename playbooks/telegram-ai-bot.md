# Маршрут реализации: Telegram AI-бот

## Когда использовать

Для Telegram-ботов с меню, callback-кнопками, AI-ответами, webhook и продуктовой логикой.

## Порядок

1. Зафиксировать основной маршрут пользователя.
2. Решить, где нужна single-message navigation.
3. Собрать callback-flow и state handling.
4. Подключить AI и внешние интеграции.
5. Пройти ручной маршрут в Telegram и подготовить webhook/deploy.
