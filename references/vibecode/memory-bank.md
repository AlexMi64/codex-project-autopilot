# Банк памяти

Храни ключевой контекст проекта в локальном memory bank, чтобы агент не переизобретал проект заново в каждой сессии.

## Правила

- вся память живет в `.codex-agent/`
- не складывай всё в один гигантский документ
- обновляй active context и progress после важных изменений
- перед планированием и реализацией сначала читай память проекта
