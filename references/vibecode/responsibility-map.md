# Карта ответственности

Объясняй проект через зоны ответственности, чтобы новичок не путал frontend, backend, базу и автоматизацию.

## Правила

- frontend отвечает за интерфейс
- backend отвечает за логику и интеграции
- database отвечает за хранение и ownership
- automation отвечает за выполнение по расписанию или цепочки действий
- deployment отвечает за env, публикацию и контроль запуска
