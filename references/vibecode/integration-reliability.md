# Надёжность интеграций

Внешние API не идеальны: они падают, дублируют события, ограничивают запросы и ломают happy path.

## Правила

- валидируй входящие payload
- логируй request id и ошибки
- retry только для временных сбоев
- думай про idempotency и дубликаты
- документируй webhook и provider secrets
