# Навигация Telegram-бота в одном сообщении

Для menu-driven Telegram-ботов используй один основной маршрут навигации через редактирование сообщения, а не поток новых сообщений.

## Правила

- меню живет в одном редактируемом сообщении
- inline-кнопки и callback-data обязательны
- для текстового ввода и истории используй обычные сообщения
- callback всегда должен подтверждаться
