# Маршрут реализации: Скрипт автоматизации

## Когда использовать

Для CLI-утилиты, парсера, синхронизатора, cron-задачи или локальной автоматизации.

## Порядок

1. Зафиксировать входы, выходы и побочные эффекты.
2. Добавить dry-run, если это уместно.
3. Реализовать минимальный happy path.
4. Добавить логи, проверки и retry для внешних вызовов.
5. Подготовить инструкцию по запуску.
